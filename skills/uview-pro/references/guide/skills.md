# Skills 能力包

## 什么是 Skills

**Skills** 是 uView Pro 提供的一套标准化、可复用、可封装的"能力包"，旨在让 AI 智能体（Agent）能够稳定、可靠地生成框架中的业务逻辑。

通过 Skills，开发者可以：
- 快速让 AI 理解 uView Pro 组件库的使用方式
- 标准化 AI 生成的代码质量
- 提升开发效率，减少重复工作

## 安装 Skills

在项目根目录运行以下命令：

```bash
pnpx skills add uview-pro/skills
```

## 开源仓库

Skills 项目完全开源，欢迎社区贡献：

安装完成后，请参阅项目中的 README 文件，了解：

- 如何自定义和扩展 Skills

## 为什么使用 Skills

### 标准化
Skills 定义了标准的组件使用模式和最佳实践，确保 AI 生成的代码符合 uView Pro 的设计规范。

### 可复用
封装好的 Skills 可以在多个项目之间共享，避免重复造轮子。

### 可扩展
开发者可以根据自己的业务需求，自定义和扩展 Skills，打造适合自己团队的 AI 辅助开发工作流。
